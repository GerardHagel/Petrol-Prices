Utworzyć plik .env na podstawie .env.example w folderze PetrolPricesAPI

W terminalu dla Backendu (PetrolPricesAPI), wpisać:

composer install

php artisan serve

W teminalu dla Frontendu (PetrolPricesClient), wpisać:

npm install

npm run dev
